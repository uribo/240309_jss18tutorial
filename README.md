### renvを使ったRパッケージのインストール

次の手順でワークショップで用いるRコードの実行に必要なパッケージをインストールします。

1. このリポジトリの内容をダウンロード
2. `240309_jss18tutorial.Rproj`を開く
3. 以下のコードを実行

```r
if (require(renv)) {
  # すべてのパッケージのインストールには
  # インターネット接続状況にもよりますが、5分程度かかります
  # Do you want to proceed? という出力が表示されたら
  # コンソールに Y と入力して Enter キーを押してください
  renv::restore()
}
```
